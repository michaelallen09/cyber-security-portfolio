### 3.2 Security Event Distribution

To understand normal activity levels, Windows Security events were reviewed and grouped by EventCode:

```spl
index="win11_endpoint_logs" sourcetype="WinEventLog:Security"
| stats count by EventCode
| sort -count
```

### 3.5 Brute Force Detection Validation (Event ID 4625)

Multiple failed logon attempts were simulated to validate detection logic.

```spl
index=win11_endpoint_logs EventCode=4625
| stats count by Account_Name, Source_Network_Address
| where count >= 5
| sort -count
```

### 3.6 Process Execution Monitoring (Sysmon)

```spl

| table _time host Image CommandLine
| sort -_time
```
