## 4. Baseline Detection Queries

With log ingestion confirmed, simple detection queries were created to demonstrate basic monitoring capability within the lab environment.

### 4.1 Failed Logon Monitoring (Event ID 4625)

To identify repeated failed authentication attempts, the following query was used:

```spl
index=win11_endpoint_logs EventCode=4625
| stats count by Account_Name, Source_Network_Address
| where count >= 5
| sort -count
```

### 4.2 Privileged Logon Visibility (Event ID 4672)

To monitor elevated sessions, privileged logon events were reviewed:

```spl
index=win11_endpoint_logs EventCode=4672
```

### 4.3 Process Execution Monitoring (Sysmon Event ID 1)

Sysmon telemetry was queried to confirm visibility of process execution activity:

```spl
index=win11_endpoint_logs "powershell.exe"
| table _time host Image CommandLine
| sort -_time
```
